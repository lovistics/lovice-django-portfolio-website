## Technical Stack

- **Backend:** Django 5.0
- **Frontend:** HTML5, CSS3, JavaScript
- **CSS Framework:** Bootstrap 4
- **Form Handling:** django-crispy-forms
- **Tagging System:** django-taggit
- **Database:** SQLite (default)
- **Image Handling:** Django ImageField
